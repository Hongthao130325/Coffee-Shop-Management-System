# Coffee Shop Management System

## Overview
The Coffee Shop Management System is a software project designed to optimize the management processes of small and medium-sized coffee shop chains, while also enabling fast and convenient online ordering.

The system helps reduce manual work, improve accuracy, and save time in tracking products, sales, revenue, and inventory management for coffee shops.

It provides an intuitive, user-friendly interface and supports future scalability as the business grows.

## Features
- User authentication (Shop Owner / Staff / Online Customers)
- Management of menu, inventory, and revenue
- Staff management, work shifts, and salary management
- Support for on-site ordering for in-store customers
- Online ordering for online customers

## Tech Stack
- Language: Python
- Frontend: HTML, CSS, JavaScript
- Backend: Django (Python)
- Architecture: MTV (Model – Template – View)
- Database: MySQL
- IDE: Visual Studio Code

## How to Run
1. Clone this repository
2. Open the project in Visual Studio Code or PyCharm
3. Create and activate a virtual environment
4. Install required dependencies
5. Configure the MySQL database connection
6. Run database migrations
7. Start the Django development server
8. Access the system via web browser
